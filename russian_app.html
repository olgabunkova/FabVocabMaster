The page could not read the topics/ folder directly. This happens when you open the HTML file with file:// or when the server doesn't expose a directory index.
Quick fix: serve the project folder with a tiny HTTP server and reload the page. Example commands:
python3 -m http.server 8000
# then open http://localhost:8000/russian_app1.html
Or with Node (if you have npm):
npx http-server .
# then open the printed URL in your browser
After serving, reload this page and topics will load automatically.
